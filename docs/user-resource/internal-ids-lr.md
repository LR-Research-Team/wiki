## Equipment
Equipment is anything that can be equipped by Lightning that influences her fighting capabilities by raising her stats or granting her passives.<br>
- `ID` is what the game uses to look up an item, or store it in a save file. These ID's are stored as `record` within `item.wdb`<br>
- `Name` is the value stored in the ZTR `txtres_us.ztr`

## Consumables
Consumables refers to anything that can be used in-battle, but that costs a resource.<br>
For potions this means it consumes the potion, for EP abilities it consumes EP-points.<br>
- `ID` is what the game uses to look up an item, or store it in a save file. These ID's are stored as `record` within `item.wdb`<br>
- `Name` is the value stored in the txtres ZTR

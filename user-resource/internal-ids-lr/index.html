    Equipment¶

    Equipment is anything that can be equipped by Lightning that influences her fighting capabilities by raising her stats or granting her passives.
    - ID is what the game uses to look up an item, or store it in a save file. These ID's are stored as record within item.wdb
    - Name is the value stored in the ZTR txtres_us.ztr

    Consumables¶

    Consumables refers to anything that can be used in-battle, but that costs a resource.

    - ID is what the game uses to look up an item, or store it in a save file. These ID's are stored as record within item.wdb
    - Name is the value stored in the txtres ZTR
